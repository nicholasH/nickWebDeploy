Another tool that I made because a friend wanted an easier way of doing things. The Steam friend game matcher is a tool to find out what steam games you have in common with your friends. Users can use this tool to figure out what multiplayer games a group of people have in common so they can all play together. The Steam Friend Game Match is limited by the steam API. If the Steam API get too many requests for informant from my server there is a chance that it will stop giving my server info. This is bad for when people have a lot of games in common.
